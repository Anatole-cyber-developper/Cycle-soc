# 🛡️ SOC Threat Monitoring Dashboard

## 📌 Description
This project is a simulated Security Operations Center (SOC) dashboard.

It monitors network activity, detects suspicious behavior, and visualizes cyber threats in real time using a custom backend API.

---

## 🚀 Features

- 📡 Real-time log monitoring (API-based)
- 🚨 Threat detection (low / medium / high)
- 📊 Live attack chart (Chart.js)
- 🌍 Attack visualization (map simulation)
- 📜 Log filtering & export (JSON)
- 🧠 Basic anomaly detection

---

## ⚙️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express)
- Visualization: Chart.js
- Hosting: Replit (API) + GitHub Pages (Frontend)
